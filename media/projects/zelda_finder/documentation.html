Noteworthy

This project utilizes JSON fetch requests to generate unique web components based on what is searched for. The application utilizes local storage to regenerate these web components through the outer HTML of each component. The project is also quite complex due the the fact that it uses unique cards based on the type of search category. Also, the project uses Firebase for a community page where the favorite counts for every card are stored.

Grading

I believe that the project currently lacks in visual appeal. However, its functionality is stable enough to warrant a good grade. The project has a large amount of time investment due to the unique card system. Therefore, I believe that despite a minor glitch with the community page, that this project deserves a strong grade.
