Functionality

This website is a search engine used to specifically find Nintendo Amiibos based on different search criteria that is presented to the user.
The user has a search box on the left side of the page that follows the screen at all times. The user can search by character name, game series,
or amiibo series. However, the user needs to be exact with their input, otherwise the API does not give results. For example, "Super Smash Bros"
will not work, but "Super Smash Bros." (with the dot) will work. The user can also select what store they are interested in buying any Amiibos
from. This option will change the link to buy each Amiibo to suit the desired store. When the search completes, the user will receive results
that contain the name, game series, Amiibo series, North American release date, and a link to buy the Amiibo for each Amiibo. The user can scroll
through their options at their leisure.

Sources

For the date selector, I used the jQueryui Datepicker. This site also uses the AmiiboAPI to search for Amiibos and their information.
This site also uses links to Amazon, Gamestop, and Walmart.
